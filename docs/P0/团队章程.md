## 一、团队简介

Camera 是一个约拍服务平台，连接有拍摄需求的用户与摄影师/化妆师，提供作品集展示、档期管理、在线预约、订单管理、评价等功能，实现约拍业务闭环。

技术栈：Vue 3 + Spring Boot + MySQL

## 五、协作流程

### 5.1 代码管理

- 主分支：`main`（稳定版本）、`dev`（开发集成）
- 功能分支命名：`feature/功能名`，修复分支：`fix/问题描述`
- 所有代码通过 Pull Request 合入 `dev`，至少一人 Review 通过
- Commit message 前缀规范：`[Human-written]` 或 `[AI-assisted]`

### 5.2 沟通机制

- 日常沟通：宿舍
- 周会：【每周两到三次，时间周四9：00-12：00，周六日9：00-12：00】
- 文档协作：GitHub + 飞书文档

### 5.3 任务管理

- 工具：GitHub Issues / 飞书多维表格
- 每阶段开始时拆分任务并分配负责人，阶段结束前进行回顾

## 六、团队公约

1. 按时参加周会，有事提前请假
