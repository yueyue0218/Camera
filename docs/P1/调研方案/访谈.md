# **“约拍到出片”软件调研访谈**

| 用户群体     | 问题模块  | 问题内容                                                   | 类型       | 备注/提示                                |
| -------- | ----- | ------------------------------------------------------ | -------- | ------------------------------------ |
| **摄影师**  | 背景信息  | 您从事摄影多久了？（引入寒暄）                                        | 开放式      | 记录经验年限和主要方向                          |
|          | 背景信息  | 平均每月约拍数量？                                              | 数值       | 了解工作量                                |
|          | 接单流程  | 您通常通过哪些渠道获取约拍订单或寻找模特？                                  | 开放式      | 可提示：社交平台、经纪公司、朋友推荐                   |
|          | 工作流复盘 | 请描述一次最近的约拍经历：接单→沟通→拍摄→修图→交付。哪个环节耗时最长或最繁琐？              | 开放式      | 可用流程图辅助标记痛点                          |
|          | 痛点挖掘  | 在履约过程中，您遇到过哪些难以解决的纠纷？（如定金、鸽子行为、版权争议、返修要求）              | 开放式      | 让受访者举具体案例                            |
|          | 工具需求  | 您目前如何管理排期、定金、素材和交付？                                    | 开放式      | 了解现有工具                               |
|          | 工具需求  | 如果有平台整合排期、定金担保和云端交付，您最看重哪些功能？请给功能排序或打分（1-5分）           | 排序       | 功能：排期管理 / 定金担保 / 云端交付 / 用户信用参考       |
|          | 核心诉求  | 对一个新平台，哪些机制能让您持续入驻并作为主要接单工具？                           | 开放式      | 可结合信用评分、评价体系、收益保障等例子                 |
| **模特**   | 背景信息  | 您是职业模特还是兼职？还是平时有拍照片的需求？拍摄类型？                           | 开放式      | 记录身份和经验                              |
|          | 现状了解  | 您平时如何筛选摄影师或接单机会？                                       | 开放式      | 可提示社交平台、口碑、朋友推荐                      |
|          | 安全与信任 | 在陌生约拍中，您最担心的问题是什么？如何规避这些风险？                            | 开放式      | 场景化，如人身安全、个人信息、拍摄质量                  |
|          | 权益保障  | 您是否遇到过照片被滥用、拖欠尾款或迟迟不出片？希望平台提供哪些保障？                     | 开放式      | 重点收集期望机制而非仅历史事件                      |
|          | 自我展示  | 您认为目前社交平台展示个人作品有哪些不足？                                  | 开放式      | 提升平台功能的参考                            |
|          | 平台期待  | 如果平台提供摄影师实名认证、信用评分、强制出片时效，您的使用意愿如何？还有哪些机制您希望看到？        | 开放式 + 打分 | 避免完全引导，可让受访者补充                       |
| **隐形受众** | 身份确认  | 您的身份：个人偶尔拍摄 / 其他                                       | 单选       | 明确分类，便于分析                            |
|          | 需求探底  | 在什么场景下，您会想找人拍照？（旅行、毕业、探店、品牌宣传等）                        | 开放式      | 收集需求场景                               |
|          | 转化障碍  | 曾经想约拍但没行动的原因是什么？请分别说明：心理因素（害怕镜头、害羞）与流程/交易障碍（价格不透明、沟通难） | 开放式      | 拆分主观与客观因素                            |
|          | 决策因素  | 浏览摄影师/模特作品时，最看重哪些信息？（审美风格、真实评价、价格、拍摄场地等）               | 开放式      | 可提示，但保持开放                            |
|          | 体验预期  | 对完全不懂摄影的普通人，您希望平台在预约-拍摄-选片过程中提供哪些引导？                   | 开放式      | 流程引导、操作简化                            |
|          | 消费意愿  | 您愿意为一次两小时周末约拍支付大概预算？                                   | 区间选择     | 提供范围：<100 / 100-300 / 300-500 / >500 |
