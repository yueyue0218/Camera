### 受访者一：有约拍需求的用户

#### 基本背景

- 身份定位：**普通约拍需求方**（非职业模特）
- 现有习惯：主要通过**社交平台**（如小红书）筛选摄影师，朋友推荐占少数

#### 核心关切

| 优先级 | 关切点 | 具体描述 |
|--------|--------|----------|
| 最高 | **拍摄风格匹配** | 个人偏好偏自然、偏抓拍的风格，而非过度精修的"网红脸"效果 |
| 中 | **人身安全与信息隐私** | 会倾向选择正规/熟系摄影师、避免偏僻地点；拍摄中不主动透露个人信息 |
| 中 | **照片版权与纠纷风险** | 担心照片在未经本人允许的情况下被摄影师用于招揽生意等商业用途 |

#### 对现有社交平台的不满

1. **同质化严重**：大量摄影师将修图重心放在"网红化"而非构图与光影本身，背离摄影初衷
2. **技术参差不齐**：构图、光影等基础技术能力仍有较大提升空间
3. **前期沟通不足**：摄影师与客户在审美偏好（明媚感 vs. 氛围感等）上缺乏有效磨合机制

#### 对平台功能的期望

- **退款或重拍权**：客户不满意时应有明确的处理渠道，摄影师和顾客的风格要求需要磨合
- **合同与条款保障**：平台应提供标准合同模板，明确照片使用授权范围，防止独立摄影师擅自滥用
- **实名认证 + 信用评分 + 强制出片时效**：三项功能均能显著提升使用意愿，关键在于摄影师档案的**信息全面程度**
- **其他机制**：暂无补充建议

### 受访者二：偏摄影师视角的潜在用户

#### 基本背景

- 身份定位：**偏向摄影师一侧**的潜在用户
- 市场认知：认为目前**缺乏成熟的约拍平台和产业生态**

#### 核心关切

| 关切点 | 具体描述 |
|--------|----------|
| **价格透明** | 市场价格不透明，自身也不了解行情，希望平台有明确的定价参考 |
| **拍摄时长** | 时长是决策关键因素之一，如"全程跟拍一整天"类需求体力消耗大，需提前明确 |
| **拍摄地点** | 地点信息需要在接单前清晰呈现 |
| **出片要求** | 需提前了解客户的出片数量、修图风格、交付格式等具体要求，避免后期纠纷 |

#### 对平台功能的期望

- **检索与筛选功能**：支持按摄影风格标签等维度快速匹配，减少双方沟通成本
- **引导机制**：平台应对新用户（无论摄影师还是客户）提供流程引导，降低上手门槛
- **市场定价参考**：希望平台提供行业价格区间，帮助摄影师和客户建立合理预期

### 受访者一：有Cosplay约拍需求的大学生

#### 基本背景

- 身份定位：**计算机科学专业大学生**（有Cosplay拍摄需求）
    
- 现有习惯：日常更多会让熟人拍摄或者自己拍，因找外部摄影师的阻力较大而经常放弃
    

#### 核心关切

| **优先级** | **关切点**    | **具体描述**                           |
| ------- | ---------- | ---------------------------------- |
| 最高      | **审美风格匹配** | 在找摄影师方面最注重对方的审美风格是否与自己的需求契合        |
| 高       | **预算与定价**  | 明确的预算限制，可接受的价格范围为2小时200元以内         |
| 中       | **沟通成本**   | 对平台存在顾虑，主要担心使用平台后仍需付出较高的沟通成本来对接摄影师 |

#### 对现有寻找方式的不满

1. **信息不透明**：目前寻找摄影师的渠道不够透明，难以准确评估对方的真实水平与报价
    
2. **沟通成本大**：前期筛选和沟通极其费时费力，导致经常直接放弃寻找陌生摄影师
    

#### 对平台功能的期望

- **降低沟通门槛**：希望平台能提供有效机制（如标签化筛选等），减少确认需求时的沟通损耗，提供类似于“拍摄企划模板”的功能，减少反复文字沟通，让摄影师一眼看懂拍摄重点。
    
- **风格与价格直观展示**：期待平台能清晰、透明地展示摄影师的代表风格与对应收费标准

### 受访者二：有旅游约拍需求的职场人士

#### 基本背景

- 身份定位：**有稳定工作的成年人**
    
- 现有习惯：倾向于在旅游的时候专门找摄影师拍照，以留下有质感的纪念
    

#### 核心关切

|**关切点**|**具体描述**|
|---|---|
|**例图真实性**|最担心遇到摄影师的“例图诈骗”（展示的样片与实际交付的客片水平严重不符）|
|**预算与定价**|具备一定的消费能力，接受的价格范围为2小时200至500元|

#### 对平台功能的期望

- **客片真实性认证：** 平台应对摄影师上传的作品进行审核，标注“真实客片”标识，甚至支持查看该摄影师近期的真实订单反馈，建设完备的评价体系。
    
- **一站式服务匹配：** 如果平台能同时推荐当地的跟妆、服装租赁，会极大地提高尝试意愿。
    
- **服务保障机制：** 针对“例图诈骗”或成片质量与预期严重不符的情况，平台应有仲裁介入和退款保障机制。
    
- **地理位置检索：** 旅游时希望能基于地图定位直接检索附近的摄影师，支持即时或短时间内预约。
