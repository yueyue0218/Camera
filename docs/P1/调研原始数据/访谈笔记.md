### 受访者一：有约拍需求的用户

#### 基本背景

- 身份定位：**普通约拍需求方**（非职业模特）
- 现有习惯：主要通过**社交平台**（如小红书）筛选摄影师，朋友推荐占少数

#### 核心关切

| 优先级 | 关切点 | 具体描述 |
|--------|--------|----------|
| 最高 | **拍摄风格匹配** | 个人偏好偏自然、偏抓拍的风格，而非过度精修的"网红脸"效果 |
| 中 | **人身安全与信息隐私** | 会倾向选择正规/熟系摄影师、避免偏僻地点；拍摄中不主动透露个人信息 |
| 中 | **照片版权与纠纷风险** | 担心照片在未经本人允许的情况下被摄影师用于招揽生意等商业用途 |

#### 对现有社交平台的不满

1. **同质化严重**：大量摄影师将修图重心放在"网红化"而非构图与光影本身，背离摄影初衷
2. **技术参差不齐**：构图、光影等基础技术能力仍有较大提升空间
3. **前期沟通不足**：摄影师与客户在审美偏好（明媚感 vs. 氛围感等）上缺乏有效磨合机制

#### 对平台功能的期望

- **退款或重拍权**：客户不满意时应有明确的处理渠道，摄影师和顾客的风格要求需要磨合
- **合同与条款保障**：平台应提供标准合同模板，明确照片使用授权范围，防止独立摄影师擅自滥用
- **实名认证 + 信用评分 + 强制出片时效**：三项功能均能显著提升使用意愿，关键在于摄影师档案的**信息全面程度**
- **其他机制**：暂无补充建议

### 受访者二：偏摄影师视角的潜在用户

#### 基本背景

- 身份定位：**偏向摄影师一侧**的潜在用户
- 市场认知：认为目前**缺乏成熟的约拍平台和产业生态**

#### 核心关切

| 关切点 | 具体描述 |
|--------|----------|
| **价格透明** | 市场价格不透明，自身也不了解行情，希望平台有明确的定价参考 |
| **拍摄时长** | 时长是决策关键因素之一，如"全程跟拍一整天"类需求体力消耗大，需提前明确 |
| **拍摄地点** | 地点信息需要在接单前清晰呈现 |
| **出片要求** | 需提前了解客户的出片数量、修图风格、交付格式等具体要求，避免后期纠纷 |

#### 对平台功能的期望

- **检索与筛选功能**：支持按摄影风格标签等维度快速匹配，减少双方沟通成本
- **引导机制**：平台应对新用户（无论摄影师还是客户）提供流程引导，降低上手门槛
- **市场定价参考**：希望平台提供行业价格区间，帮助摄影师和客户建立合理预期

#### 核心关切

| **优先级** | **关切点**    | **具体描述**                           |
| ------- | ---------- | ---------------------------------- |
| 最高      | **审美风格匹配** | 在找摄影师方面最注重对方的审美风格是否与自己的需求契合        |
| 高       | **预算与定价**  | 明确的预算限制，可接受的价格范围为2小时200元以内         |
| 中       | **沟通成本**   | 对平台存在顾虑，主要担心使用平台后仍需付出较高的沟通成本来对接摄影师 |

#### 对现有寻找方式的不满

1. **信息不透明**：目前寻找摄影师的渠道不够透明，难以准确评估对方的真实水平与报价
    
2. **沟通成本大**：前期筛选和沟通极其费时费力，导致经常直接放弃寻找陌生摄影师
    

#### 对平台功能的期望

- **降低沟通门槛**：希望平台能提供有效机制（如标签化筛选等），减少确认需求时的沟通损耗，提供类似于“拍摄企划模板”的功能，减少反复文字沟通，让摄影师一眼看懂拍摄重点。
    
- **风格与价格直观展示**：期待平台能清晰、透明地展示摄影师的代表风格与对应收费标准

### 受访者一：有约拍需求的普通用户

#### 基本背景

- 身份定位：**普通拍摄需求方**（非职业模特），主要为日常拍照、个人写真
- 现有习惯：主要通过**小红书**筛选摄影师，合作前会查看样片判断风格和水平

#### 核心关切

| 优先级 | 关切点 | 具体描述 |
|--------|--------|----------|
| 最高 | **成片质量** | 最担心拍摄质量不达标、成片效果差，以成片满意度作为是否付费的核心标准 |
| 高 | **人身安全** | 尤其担心与陌生摄影师线下接触的安全问题，会采取多种方式自我保护 |
| 中 | **肖像使用** | 允许摄影师将成片用于宣传或作品集展示，对此无明显抵触 |

#### 风险规避措施

| 措施 | 具体做法 |
|------|----------|
| 时间选择 | 尽量安排在白天拍摄 |
| 接触边界 | 不与摄影师单独吃饭，保持正常社交距离 |
| 人群偏好 | 优先选择校内摄影师，更熟悉、更有安全感 |
| 主动姿态 | 会主动购买饮料等缓和氛围，但保持分寸 |

#### 对现有平台的不满

1. **样片与实际差距大**：精修过度的样片容易误导选择，无法真实反映摄影师水平
2. **原片与修图不区分**：现有平台未清晰区分摄影师原片（直出未修）与修图后成片

#### 对平台功能的期望

- **质量审核与退款机制**：希望平台设立审核团队监督拍摄质量，成片不达标时支持退款；对付费顺序（先付/后付）无强烈偏好，核心诉求是"不满意可退"
- **作品展示分级**：平台应清晰区分原片与精修成片，方便用户真实判断摄影师水平
- **实名认证**：非常认可该机制，认为能显著提升安全感和合作意愿
- **信用评分 + 出片时效**：支持建立信用评分体系，同时希望强制约定出片时效，避免摄影师拖延交片
- **双向互评**：用户可评价摄影师，摄影师也可评价用户，形成双向约束
- **激励机制**：建议对摄影师和拍摄者双方设置激励，提升平台活跃度

### 受访者二：摄影师视角的用户

#### 基本背景

- 身份定位：**兼职摄影师**，主要拍摄类型为人像写真
- 接单现状：主要通过**小红书发布作品**和**朋友推荐**获取约拍订单，月均约拍 2-3 次
- 现有工具：使用微信沟通排期，无专门的管理工具

#### 核心关切

| 优先级 | 关切点 | 具体描述 |
|--------|--------|----------|
| 最高 | **客户爽约/鸽子行为** | 多次遇到客户临时取消或迟到，浪费大量准备时间，缺乏违约惩罚机制 |
| 高 | **定金与尾款纠纷** | 部分客户以不满意为由拒付尾款，缺乏第三方裁定机制 |
| 高 | **沟通成本高** | 前期风格沟通反复耗时，客户往往无法清晰表达需求，导致拍摄方向偏差 |
| 中 | **版权争议** | 个别客户私自将照片用于商业用途（如店铺宣传），未经授权 |

#### 当前工作流程与痛点

| 环节 | 现状 | 痛点 |
|------|------|------|
| 接单 | 小红书私信/朋友推荐 | 被动等待，难以稳定获客 |
| 沟通 | 微信文字+参考图 | 反复沟通效率低，客户需求模糊 |
| 排期 | 手动记在日历/备忘录 | 容易遗漏，无法对外展示空闲时段 |
| 拍摄 | 线下当面 | 客户迟到/爽约无约束 |
| 修图 | Lightroom + Photoshop | 客户返修要求不明确，反复修改 |
| 交付 | 百度网盘/微信传输 | 文件管理混乱，无法追踪交付状态 |

#### 对平台功能的期望

- **排期管理**：最看重，希望平台提供可视化档期管理，客户能直接看到空闲时段并预约，减少沟通成本（重要性打分：5/5）
- **定金担保**：希望平台提供第三方资金托管，预约时收取定金，完成后自动结算尾款；客户爽约时定金归摄影师（重要性打分：5/5）
- **云端交付**：支持在线选片和交付，客户确认后自动标记完成，避免文件传输混乱（重要性打分：4/5）
- **信用评分参考**：希望能看到客户的历史评分，筛选靠谱客户，避免遇到频繁爽约或恶意拒付的用户（重要性打分：4/5）
- **风格标签与匹配**：希望平台支持摄影风格标签（如日系、胶片、清新、暗调等），让客户能按偏好找到匹配的摄影师，减少前期沟通摩擦
- **标准化合同**：平台应提供合同模板，明确拍摄数量、修图张数、交付时限、版权归属等条款，减少纠纷

#### 持续入驻的关键因素

- **稳定的客源流量**：平台需有足够用户基数，能带来持续订单
- **合理的抽成比例**：平台佣金不宜过高，建议不超过 10-15%
- **公平的评价体系**：双向评价且不可随意删除，防止恶意差评
- **摄影师成长体系**：如等级认证、优质摄影师推荐位等激励机制

## 受访者一：有约拍需求的用户

**基本背景**
* **身份定位**：普通约拍需求方（在校大学生） 
* **现有习惯**：目前主要拉着朋友或认识的人一起拍照；由于缺乏官方、正规的保证渠道，现阶段不会刻意尝试约陌生摄影师

---

**核心关切**

| 优先级 | 关切点 | 具体描述 |
| :--- | :--- | :--- |
| **最高** | **人身与财产安全** | 担心单独与陌生人出去的人身安全问题，以及交易过程中的财产风险 |
| **最高** | **信用与品质保障** | 极其看重摄影师的信用评级和实名认证，要求好评率公开透明，确保钱花在实处 |
| **中** | **信息对称与透明** | 希望平台能打破中间商的信息差，真实反映摄影师的专业水平、态度及出片时效  |
| **中** | **照片滥用与纠纷** | 担心照片被摄影师滥用或摄影师拖延出片，希望有标准来保证消费者合法权益 |

---

**对现有社交平台的不满**
1.  **安全感缺失**：缺乏官方背景和健全的保障机制，导致用户对陌生约拍存在较大的心理隔阂
2.  **反馈机制效率低**：现有的某些App存在客服反馈不及时、用户投诉无果的情况
3.  **信息不对称**：难以低成本地获取摄影师真实的信用状况、专业手法及服务态度
4.  **风格固化倾向**：担心大数据算法会导致信息茧房，使用户难以接触到多元化的新风格 

---

**对平台功能的期望**
* **健全的评价与举报体系**：评价必须公开（包含差评），设立恶意差评举报机制，且客服反馈需高效及时 
* **官方交易保护**：偏向于通过平台直接交款作为中间保障，以便在出现纠纷时留存可靠证据 
* **实名认证 + 信用评分 + 出片时效限制**：这些功能是提升使用意愿的核心，能达到“信息对称”的安全感 
* **其他机制**：
    * **通俗易懂的沟通**：摄影师描述应口语化，避免使用令普通人费解的专业术语
    * **灵活的引流策略**：初期通过免费优惠、新手适应期吸引用户；针对不同预算区间进行精准推送
    * **双向保护机制**：在保护消费者的同时，也应通过违禁词过滤等手段保护摄影师免受辱骂
