# Camera 约拍服务平台

## 项目简介
Camera 是一个约拍服务平台，连接有拍摄需求的用户与摄影师/化妆师，
提供作品集展示、档期管理、在线预约、订单管理、评价等功能，实现约拍业务闭环。
